Omniauth for [Thebase](https://thebase.com/)

Instration
----------

Add this line to your application's Gemfile:

    gem 'omniauth-thebase'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install omniauth-thebase

Test
----

```
bundle exec rspec
```
